# متجر روقان الإلكتروني - Rawqan Bookstore

## الوصف
متجر كتاب إلكتروني عربي فاخر مع تصميم Liquid Luxury. يدعم التسوق، السلة، الدفع، وتتبع الطلبات.

## المميزات
- 🎨 تصميم سينمائي فاخر بالألوان: أسود عميق، ذهبي سائل، أبيض ناعم
- 📱 متجاوب مع جميع الأجهزة
- 🛒 سلة تسوق متكاملة
- 💳 نظام دفع وتتبع طلبات
- 🔄 نظام مزامنة بين الأجهزة (عند تشغيل خادم Node.js)
- 🎫 نظام كوبونات خصم
- 📊 لوحة تحكم إدارية كاملة

## التقنيات المستخدمة
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js + Express (اختياري للمزامنة)
- **تخزين البيانات**:
  - `localStorage`: للوضع المحلي (بدون خادم)
  - ملف JSON: للوضع المزامن (مع خادم Node.js)

## كيفية التشغيل

### الوضع المحلي (بدون خادم - لاستضافة ثابتة)
1. افتح ملف `index.html` مباشرة في المتصفح
2. يعمل تلقائياً باستخدام `localStorage`

### الوضع المزامن (مع خادم Node.js - للمزامنة بين الأجهزة)
1. تأكد من تثبيت Node.js
2. افتح Terminal في مجلد `server`:
   ```bash
   cd server
   npm install
   npm start
   ```
3. افتح المتصفح على: `http://localhost:3000`

## لوحة التحكم الإدارية
- كلمة المرور الافتراضية: `admin123`
- يمكنك إدارة: الكتب، المؤلفون، الفئات، البنرات، الكوبونات، الطلبات

## العملة
- ريال سعودي (SAR) - افتراضي
- ريال يمني (YER) - قابل للتبديل
- سعر الصرف: 1 SAR = 140 YER
