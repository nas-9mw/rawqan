# 📚 متجر روقان - نظام الخادم المركزي

## المشكلة التي تم حلها

**قبل:**
- كانت البيانات تُحفظ في `localStorage` لكل جهاز بشكل منفصل
- أي تعديل من لوحة التحكم يظهر فقط على الجهاز نفسه
- كل جهاز لديه نسخة مختلفة من البيانات

**بعد:**
- جميع البيانات تُحفظ في ملف مركزي واحد على الخادم
- أي تعديل من لوحة التحكم يظهر فوراً على **جميع الأجهزة** المتصلة
- جميع الأجهزة تشترك في نفس البيانات

---

## 🚀 كيفية تشغيل الخادم

### 1. المتطلبات
تحتاج إلى تثبيت **Node.js** على جهاز الخادم:
- حمل من الموقع الرسمي: https://nodejs.org/
- اختر النسخة **LTS** (النسخة المستقرة)

### 2. تثبيت المكتبات
افتح Terminal (أو PowerShell في ويندوز) واذهب لمجلد الخادم:

```bash
cd "C:\Users\DR-MARWAN\Desktop\شراع\عملاء\روقان\rawqan-site\روقان\server"
```

ثم قم بتثبيت المكتبات:

```bash
npm install
```

### 3. تشغيل الخادم

```bash
npm start
```

أو لأغراض التطوير (مع إعادة التشغيل التلقائي):

```bash
npm run dev
```

---

## 🌐 كيفية الوصول للمتجر

### من نفس الجهاز (الخادم):
افتح المتصفح واذهب إلى:
```
http://localhost:3000
```

### من أجهزة أخرى في نفس الشبكة:
1. اعرف عنوان IP الخاص بجهاز الخادم:
   - في ويندوز: افتح CMD واكتب `ipconfig`
   - ابحث عن `IPv4 Address` مثلاً: `192.168.1.100`

2. من أي جهاز آخر في الشبكة، افتح المتصفح واذهب إلى:
```
http://192.168.1.100:3000
```

(استبدل `192.168.1.100` بعنوان IP الخاص بالخادم)

---

## 📁 ملف البيانات المركزي

جميع البيانات تُحفظ في ملف واحد:
```
server/data.json
```

هذا الملف يحتوي على:
- الكتب
- المؤلفين
- الفئات
- البنرات
- الكوبونات
- الطلبات
- التقييمات

### ملاحظة هامة:
**لا تحذف ملف `data.json`** - فهو يحتوي على جميع بياناتك. قم بعمل نسخة احتياطية منه بشكل دوري.

---

## 🔧 ملفات الخادم

```
server/
├── package.json    ← إعدادات المشروع والمكتبات
├── server.js       ← الكود الخاص بالخادم وجميع الـ API
└── data.json       ← ملف البيانات المركزي (يُنشأ تلقائياً)
```

## 📝 ملخص سريع

1. **ثبت Node.js**
2. **اذهب لمجلد server**: `cd server`
3. **ثبت المكتبات**: `npm install`
4. **شغل الخادم**: `npm start`
5. **افتح المتصفح**: `http://localhost:3000`

أي تعديل تقوم به من لوحة التحكم سيظهر فوراً على جميع الأجهزة المتصلة بالخادم! 🎉
